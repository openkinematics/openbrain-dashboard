# Attribution

OpenBrain Dashboard is an OpenKinematics project for operating robots that run `openbrain-ros`.

Project foundations:

- OpenKinematics design tokens (electric blue `oklch(0.55 0.19 250)`, Orbitron + Geist fonts, sharp corners)
- Local-first preferences and fleet registry
- Responsive operator workflows for CockPit, Maps, Missions, Health, and My UI
- WebRTC video subscription integrated with `openbrain-ros` streamer
